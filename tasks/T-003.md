# T-003 · 森林舞台分层场景 `prototype/forest-stage.html`

**负责**：Coder（实现）→ Reviewer（评审）  **依赖**：T-001 style-lab 渲染引擎  **分支**：`feature/T-003-forest-stage`

---

## 背景

T-001 style-lab 的渲染管线已验证可用（分形树、贝塞尔松、丛林、雾、萤火、呼吸、视差）。T-003 把它抽成一个**纯沉浸场景**——没有控制面板，只有森林在呼吸。

## 范围

一个**单文件** `prototype/forest-stage.html`：

- 4–6 层视差森林剪影（带分形树/松树/丛生树/棕榈/灌木，默认 5 层）
- 2–3 层雾漂移（默认 3 层）
- 萤火粒子（正弦漂移 + 发光）
- 6s 呼吸缩放律动
- 默认竖屏（9:16），支持全屏切换
- 淡入开场（从黑到亮，约 1.5s）
- 触摸倾听者树洞有微光反馈

## 技术约束

- 纯 HTML/CSS/canvas，零构建
- 复用 style-lab 的 `CONFIG` 结构（单一真相源，默认 青雾）
- 复用已验证的渲染函数（分形树/有机松树/丛生树/棕榈/叶/灌木）
- 遵守风格宪法禁止项

## 不在范围

无控制面板、无 JSON 导出、无预设切换。这就是一扇森林窗户。

## 验收标准

1. 双击本地打开，淡入显示森林场景（非黑屏）
2. headless Chrome 截图验证：非纯黑、非空白
3. 60fps 流畅运行，呼吸缩放可感知
4. 触摸树洞有涟漪 + 萤火聚拢反馈
5. 窗口缩放不破图
6. prefers-reduced-motion 生效
